## Smoke Command
```bash
php tests/Chaos/run_scenario.php --duration-ms=1200
```

Expected: banner + `Response: Success` captured and `result=PASS`.

## Scenario Command
```bash
php tests/Chaos/run_scenario.php --scenario=docs/ami-client/chaos/scenarios/s1-permission-denied.json --duration-ms=1500
```

## Supported Script Step Types
- `send_banner`
- `send_frame`
- `send_on_action`
- `send_garbage`
- `close`

## Script Controls
- `at_ms` (timed scheduling)
- `delay_ms` (write delay)
- `delimiter` (`\\r\\n\\r\\n` or `\\n\\n`)
- `chunk_size` + `chunk_delay_ms` (partial writes)
